TSM

Tribestream Server Management is a solution to provision your server through SSH quickly and easily from the Java Virtual Machine to your Application in a reliable and automated way.

Installation

$ mkdir -p /opt/tsm
$ unzip tsm-LAST-distribution.zip -d /opt/tsm
$ export PATH=/opt/tsm/tsm-LAST-distribution/bin:$PATH
$ tsm -i
Using tsm.home=/opt/tsm/tsm-0.2 | java=/opt/java/jdk1.8.0_45.jdk/bin/java

tsm @ 0.2$

Fully Java

TSM is developped in Java 8 and then portable on all your boxes.

Targetting UNIx Servers

TSM is compatible with most of UNIx servers which is the most used platform for java applications.

PCI friendly

Being a remote deployment orchestrator TSM is compatible with PCI being usable on any jumpbox!
